Frontend

HTML

تعد HTML (HyperText Markup Language) لغة توضيح النصوص المستخدمة لبناء صفحات الويب. تتكون HTML من علامات (tags) تعرف الهيكل والمحتوى لكل عنصر على الصفحة، مثل النصوص، الصور، والروابط. تُعتبر HTML أساسية لتصميم وتنسيق محتويات الصفحة، وتسهل عرضها بشكل منظم على المتصفحات المختلفة.

ابدأ تعلمها

CSS

CSS (Cascading Style Sheets) هي لغة تنسيق النمط التي تستخدم لتعديل مظهر عناصر صفحات الويب التي تم إنشاؤها بواسطة HTML. تُمكن CSS من تعيين الألوان، الخطوط، الهوامش، والتنسيقات الأخرى لتحسين تجربة المستخدم وجعل الصفحة أكثر جاذبية وسهولة في القراءة.

ابدأ تعلمها

JavaScript

JavaScript هي لغة برمجة تُستخدم لإضافة تفاعل وديناميكية إلى صفحات الويب التي تم إنشاؤها باستخدام HTML وتنسيقها بواسطة CSS. تُساعد JavaScript في إضافة الوظائف التفاعلية مثل التحقق من النماذج، والتحكم في العناصر، وتغيير المحتوى بناءً على تفاعل المستخدم، مما يُحسّن تجربة المستخدم ويجعل الصفحات أكثر ديناميكية وتفاعلية.
